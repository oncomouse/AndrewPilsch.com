# About This Generator

A lightweight static site generator built with Python, designed for developers who want a fast, minimal website without unnecessary complexity.

## Philosophy

No JavaScript. No CSS frameworks. No build complexity. Just markdown, templates, and clean HTML. Inspired by the minimal, monospace aesthetic of modern developer sites and AI company websites.

## Technology

- **Python 3** - Fast, readable, minimal dependencies
- **Markdown** - Simple content format
- **Jinja2** - Flexible, powerful templating
- **Watchdog** - Real-time file watching for development

## Design Goals

- **Content-First** - Focus on what you're writing, not how it looks
- **Minimal** - No unnecessary features or configuration
- **Fast** - Instant builds and lightweight output
- **Professional** - Beautiful without being flashy
- **Developer-Friendly** - Simple command-line interface

Build your site. Deploy anywhere. Done.
